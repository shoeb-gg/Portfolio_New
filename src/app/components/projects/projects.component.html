Portfolio
(Angular Universal, TailwindCSS NestJS, Prisma)
This Web App is built using cutting-edge technologies like Angular v16, Angular Material, and Reactive Forms for a seamless user experience. With TailwindCSS, it boasts a responsive design and snappy user interactions. Server-Side Rendering is achieved by implementing Angular Universal which helps in enhancing performance and SEO. I incorporated motion graphics with ngx-lottie component. To ensure fast and secure backend operations, I used NestJS with Fastify Adapter, Rate Limiting, and Helmet library. Supabase is used as the database to save messages. Google Analytics tracks page visits, while captivating animations are achieved with anime.js and aos.
